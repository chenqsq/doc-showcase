## 当前固定口径

- 平台统一学习结构为：`学科大类 -> 学科 -> 阶段 -> 模块 -> 课节 -> 状态`
- 平台先解决“学习如何被持续组织与推进”
- 子引擎再解决“这一节具体怎么教、怎么练、怎么评”
- 学科层负责“这一门课接入时要交什么接口和策略资产”
