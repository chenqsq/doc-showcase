# AI教师智能体群引擎 PRD

> 文档层级：子引擎层  
> 文档目的：定义 AI教师智能体群引擎在比赛版系统中的职责、能力边界和验收重点  
> 核心结论：这不是单 Agent 的答题器，而是一组协同负责诊断、重规划、讲解、评分、笔记生成和资料入库的教学引擎

## 1. 子引擎目标

AI教师智能体群引擎固定承担 6 类正式目标：

1. 生成初始学习地图
2. 执行短诊断并校准起点
3. 在学习中实时重规划
4. 完成讲解、作答、评分和成长反馈
5. 生成思维导图和结构化笔记
6. 处理资料识别、结构化和知识进化

## 2. 能力分段

| 分段 | 说明 | 输出 |
| --- | --- | --- |
| 开学装配段 | 根据选科和历史记录生成初始地图 | `AI学习地图` |
| 诊断校准段 | 执行短诊断并校准主线 | `重规划事件` |
| 闯关教学段 | 讲解、提问、作答、评分和下一步推进 | `闯关学习任务`、`成长反馈事件` |
| 实时重规划段 | 在卡点、遗忘和兴趣下滑时调整路线 | 新节点与回主线条件 |
| 复习沉淀段 | 生成笔记、思维导图和复习计划 | `复习笔记包` |
| 知识进化段 | 处理新资料并更新知识资产 | `知识资产包`、`知识演化记录` |

## 3. 核心功能需求

| 编号 | 能力 |
| --- | --- |
| `FR-01` | 初始地图生成 |
| `FR-02` | 短诊断校准 |
| `FR-03` | 实时重规划 |
| `FR-04` | 流式教学与评分 |
| `FR-05` | 成长反馈输出 |
| `FR-06` | 学习画像增量更新 |
| `FR-07` | 思维导图与结构化笔记生成 |
| `FR-08` | 资料识别与自动入库 |
