# AI教师智能体群引擎 教学策略设计

> 文档层级：子引擎层  
> 文档目的：定义智能体群如何沿学习地图推进、补桥、激励和沉淀笔记  
> 核心结论：教学策略的重点不是多说，而是知道什么时候推、什么时候补、什么时候给成长感、什么时候沉淀复习资产

## 1. 策略主线

- 地图先于问答
- 当前关卡先于自由发挥
- 回补优先于硬推
- 正反馈优先于冷反馈
- 沉淀优先于遗忘

## 2. 教学策略要点

| 场景 | 策略 |
| --- | --- |
| 学生刚进入 | 给初始地图和第一关，不先把人丢进空白聊天页 |
| 短诊断后 | 重排起点和主线顺序 |
| 连续错误 | 打开补桥支线并解释原因 |
| 达标通过 | 给出成长反馈并解锁新区域 |
| 一轮结束 | 输出结构化笔记 |
| 阶段结束 | 输出思维导图与复习计划 |
