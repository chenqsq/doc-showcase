# 知脉课堂访问与评测手册

> 文档层级：交付文档  
> 文档目的：说明评委如何访问作品、如何登录、如何按步骤体验核心链路  
> 核心结论：访问手册必须让评委在不依赖现场口头解释的情况下，也能完成一次标准体验

## 1. 访问方式总览

建议正式交付时提供两种访问入口：

| 入口 | 用途 |
| --- | --- |
| 前端工作台访问地址 | 评委直接体验作品页面 |
| ADP 官方体验链接 | 作为智能体原生能力与兜底演示入口 |

## 2. 交付前必须替换的字段

下表在正式提交前替换为真实值：

| 字段 | 当前填写方式 |
| --- | --- |
| 工作台访问地址 | `待填写：如 GitHub Pages / 服务器域名` |
| ADP 体验链接 | `待填写：腾讯 ADP 发布链接` |
| 演示学生账号 | `待填写` |
| 演示教师账号 | `待填写` |
| 管理员账号 | `待填写` |
| 默认课程 | `高等数学` |
| 默认章节 | `导数与微分` 或演示用章节 |

## 3. 推荐评委体验路径

1. 打开工作台首页
2. 进入课堂资料上传与重构页，查看已准备好的课堂重构结果
3. 进入学生伴学会话页，输入演示问题或拍题
4. 查看作答后的掌握度与错题标签
5. 进入错题与复习页，查看变式题与复习计划
6. 进入教师洞察页，查看班级趋势与补讲建议

## 4. 角色与权限说明

| 角色 | 可访问内容 |
| --- | --- |
| 学生 | 首页、伴学会话、错题与复习 |
| 教师 | 首页、教师洞察、课堂重构结果 |
| 管理员 | 全部页面，含配置与访问管理 |

## 5. 操作说明

### 5.1 学生评测路径

- 使用学生账号登录
- 进入伴学会话页
- 使用默认演示问题或上传题图
- 完成一轮作答并查看掌握度快照

### 5.2 教师评测路径

- 使用教师账号登录
- 打开教师洞察页
- 查看高频错因、风险学生和补讲建议

### 5.3 管理评测路径

- 使用管理员账号登录
- 打开管理与接入配置页
- 查看课程配置、访问凭证与接入参数

## 6. API 摘要

正式提交时建议附上这几组接口摘要：

- 素材上传与解析
- 知识重构查询
- 学习会话创建
- 流式伴学对话
- 作答提交与评分
- 教师洞察查询
- 管理配置与访问校验

详细定义见 [06-接口与API说明.md](./06-接口与API说明.md)。

## 7. 异常说明

| 场景 | 说明 |
| --- | --- |
| 工作台暂时不可用 | 可切换到 ADP 官方体验链接进行体验 |
| 学生账号异常 | 使用手册中备用演示账号 |
| 图片识别失败 | 使用预置题图样例 |
| 教师洞察为空 | 使用已有演示班级数据 |

## 8. 评测前检查清单

- 工作台地址可访问
- ADP 体验链接可访问
- 三类账号可登录
- 默认课程与默认章节已配置
- 标准演示链路数据已准备完成
- 视频与 PPT 中展示的页面和线上页面一致
