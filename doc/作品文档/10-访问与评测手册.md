# AI主导学习生命周期的自进化自学智能体平台访问与评测手册

> 文档层级：作品主文档  
> 文档目的：让评委能独立进入系统并完成主流程体验  
> 核心结论：评测应先走学生主线，再补看后台自治能力

## 2. 推荐评测顺序

1. 使用学生账号进入前端
2. 选择高等数学，点击“开始学习”
3. 查看 AI 初始学习地图
4. 完成短诊断，观察地图校准
5. 进入当前关卡并完成一轮作答
6. 查看补桥提示、通关反馈和画像变化
7. 打开笔记复习与成长页，查看思维导图和结构化笔记
8. 再进入后台，查看资料注入和知识演化记录

## 3. 学生主线评测路径

### 3.1 开学体验

- 选择高等数学
- 点击“开始学习”
- 确认已生成初始地图和当前推荐关卡

### 3.2 地图与闯关体验

- 完成短诊断
- 观察地图是否重排
- 进入当前关卡
- 提交一次正确或错误作答
- 观察即时反馈、补桥提示和地图推进

### 3.3 笔记与成长体验

- 打开笔记复习与成长页
- 查看思维导图
- 查看结构化笔记
- 查看画像变化和成长曲线

## 4. 后台评测路径

### 4.1 资料注入与知识库演化后台

- 上传一份文档、图片或题单
- 查看识别状态
- 查看知识资产包
- 查看演化记录
- 查看影响到哪些地图节点

### 4.2 系统自治与策略分析后台

- 查看 Agent 状态
- 查看策略快照
- 查看重规划日志
- 查看画像更新日志
- 查看异常与审计信息

## 5. API 摘要

重点接口组：

- 选科与开学
- 学习地图生成与短诊断
- 实时地图状态与重规划事件
- 闯关学习会话
- 作答评分与画像更新
- 笔记 / 思维导图 / 复习计划
- 资料注入与自动入库
- 系统自治与策略分析

## 6. 常见异常说明

| 异常 | 处理方式 |
| --- | --- |
| 地图未生成 | 刷新后重试，或切换到默认高数演示地图 |
| 流式讲解异常 | 使用缓存讲解结果完成演示 |
| 笔记未刷新 | 重新打开笔记页或查看上一次演示样例 |
| 新资料识别失败 | 使用准备好的备用资料包 |
| 后台日志为空 | 切换到已有演示记录 |

## 7. 评测通过标准

- 学生能完整走通一次从开学到通关的学习流程
- 能看到至少一次实时重规划
- 能看到至少一次画像更新
- 能看到至少一份思维导图或结构化笔记
- 能看到新资料入库和知识演化记录
