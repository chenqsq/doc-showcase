# 知脉课堂演示视频脚本

> 文档层级：交付文档  
> 文档目的：给出 3 分钟内可直接执行的视频脚本、镜头顺序与口播要点  
> 核心结论：视频要跑完整条标准链路，节奏快、镜头稳、口播少而准

## 1. 视频总要求

- 总时长：`180` 秒内
- 建议分辨率：`1920x1080`
- 建议结构：问题引入 `15s` + 核心链路 `135s` + 总结 `30s`

## 2. 时间轴脚本

| 时间 | 画面 | 口播 / 字幕 |
| --- | --- | --- |
| `0s-15s` | 高校课堂资料碎片化场景切换：PPT、板书、录音、拍题截图 | “课堂资料分散、课后复盘低效、教师难看学情，是高校课堂的典型问题。” |
| `15s-30s` | 打开知脉课堂首页 | “知脉课堂把课堂资料上传、知识重构、学生伴学、错题复练和教师洞察整合到一个工作台中。” |
| `30s-55s` | 在上传与重构页上传录音、PPT、板书照片 | “系统支持多模态课堂素材统一解析，自动生成课堂素材包。” |
| `55s-80s` | 展示知识重构结果：重点、难点、关系图、课堂摘要 | “课堂内容会被重构为结构化知识结果，方便学生复盘，也方便教师回看。” |
| `80s-115s` | 进入学生伴学页，拍题提问，流式返回讲解和提示 | “学生可以基于当前课堂上下文直接提问，系统流式讲解步骤，并给出下一步提示。” |
| `115s-140s` | 学生提交作答，出现掌握度快照和错题标签 | “作答后，系统会计算掌握度，识别风险，并沉淀错题画像。” |
| `140s-160s` | 打开错题与复习页，展示变式题和复习计划 | “针对错题，系统自动生成同类变式，并安排间隔复习任务。” |
| `160s-175s` | 打开教师洞察页，展示班级趋势、风险学生、补讲建议 | “教师端可查看群体高频错因、风险学生和补讲建议。” |
| `175s-180s` | 回到首页或封面收尾 | “知脉课堂，让课堂知识真正流动起来。” |

## 3. 拍摄建议

- 屏幕录制统一使用桌面端工作台
- 关键交互处放大鼠标或高亮点击区域
- 知识重构结果和教师洞察页建议保留 2 到 3 秒静止镜头
- 伴学会话页要保留一小段流式输出过程，避免像静态截图

## 4. 字幕建议

全片只保留 5 组关键词字幕：

1. 多模态课堂素材接入
2. 课堂知识自动重构
3. 学生上下文伴学
4. 错题画像与变式复练
5. 教师学情洞察
